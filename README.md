# coding-task-app-nodeServer

Set up steps:
1. Clone the repo 
2. Install all dependencies with npm i
3. Run the server with the comand in the terminal: node server.js 
4. Open http://localhost:3000/test
5. Open http://localhost:3000/client-data
6. Go to Postman or Insomnia app and put the link http://localhost:3000/api/v1/parse 
and in Body:  {"data": "JOHN0000MICHAEL0009994567"} and you will see the result
7. Go to Postman or Insomnia app and put the link http://localhost:3000/api/v2/parse 
and in Body:  {"data": "JOHN0000MICHAEL0009994567"} and you will see another result
